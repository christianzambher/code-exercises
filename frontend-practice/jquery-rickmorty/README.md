# jQuery Rick & Morty Explorer

Aplicación frontend desarrollada con jQuery que consume la API de Rick & Morty para buscar y visualizar personajes.

## 🧠 Tecnologías

* jQuery
* JavaScript
* HTML
* CSS

## ⚙️ Funcionalidades

* 🔍 Búsqueda de personajes
* 📋 Listado dinámico
* 🔄 Paginación básica
* ⚡ Consumo de API con AJAX

## 📦 Instalación

Abrir el archivo `index.html` en el navegador.

## 📌 Nota

Este proyecto tiene como objetivo practicar manipulación del DOM y consumo de APIs utilizando jQuery.
