## Code-Exercises CODEIGNITER 4

Ejercicios y practicas del lenguaje para el entendimiento y compresion de CodeIgniter 4.

## 🧩 Commands
### Crear proyecto
```
composer create-project codeigniter4/appstarter [NOMBRE_PROYECTO]
```
### Crear Controlador
```
php spark make:controller [NOMBRE_CONTROLADOR]
```
### Crear Modelo
```
php spark make:model [NOMBRE_MODELO]
```
### Correr proyecto
``` 
# dentro de la carpeta del proyecto
cd [UBICACION PROYECTO]
php spark serve
```
## 🧠 Concepts
### Modelos
Los modelos se ubican en la carpeta de app/Models
### Vistas
Las vistas se ubican en la carpta de app/Views
### Registrar rutas
Agregar las rutas en el archivo de app/Config/Routes.php
### Cambiar a Modo Desarrollo
Hacer una copia de env y renombralo como .env
